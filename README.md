# SmashLab🎾
A full-stack e-commerce web application built with **Next.js, MongoDB, and Iron Session**, focusing on real-world user flows, including:

- Guest cart management
- Authentication-based cart synchronization
- Checkout flow simulation (mock payment gateway)
- GA4 ecommerce event tracking

This project is designed to simulate a production-level e-commerce system with a focus on user journey, state consistency, and analytics tracking.

## 🚀Live Demo
https://smashlab-six.vercel.app/

## 🧰Tech Stack
### Frontend
- Next.js (App Router)
- TypeScript
- React Query
- Zustand

### Backend
- Next.js API Routes
- MongoDB + Mongoose

### Authentication
- Iron Session (cookie-based session handling)

### Analytics
- Google Analytics 4 (GA4 Ecommerce Events)
